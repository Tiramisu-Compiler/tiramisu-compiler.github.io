std::string  name 
)
inline

Construct an expression that represents a variable.

type is the type of the variable and name is its name. If a variable with the same name has previously been declared, but with a different type, this constructor will fail. That way two variables with the same name are necessarily equal.

tiramisu::var::var ( std::string  name)
inline

Construct an expression that represents an untyped variable.

For example to declare the variable "t", use tiramisu::var("t"); If a variable with the same name has previously been declared, this object will have the same type (i.e. it will be equal to the other variable object).

tiramisu::var::var ( std::string  name,
expr  lower_bound,
expr  upper_bound 
)
inline

Construct a loop iterator that has name as a name.

lower and upper are expressions that represent the lower and upper bounds of this iterator. For example, the iterator i in the following for loop

for (i = 0; i < 10; i++)

can be declared as

var i("i", expr(0), expr(10));
